# Digital Growth Studio Website

This ZIP contains a complete professional website for Aissam and Ismaeil.

## Files

- `index.html` — main website page
- `assets/css/styles.css` — full responsive design
- `assets/js/script.js` — mobile navigation
- `assets/img/team/` — replace the placeholder images with your personal photos
- `assets/img/portfolio/` — creative thumbnails for website and data/marketing projects
- `assets/img/decor/` — hero image

## Replace personal photos

Put your real photos in:

- `assets/img/team/aissam-placeholder.jpg`
- `assets/img/team/ismaeil-placeholder.jpg`

You can either keep the same filenames or update the filenames in `index.html`.

## Contact form

The current form uses a placeholder Formspree link:

`https://formspree.io/f/your-form-id`

Create a free Formspree form and replace this link, or replace the form with your email/WhatsApp link.

## GitHub Pages

Upload all files to a GitHub repository, then activate GitHub Pages from Settings > Pages.
